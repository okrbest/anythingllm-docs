# 벡터 데이터베이스

TeamplGPT는 [LanceDB](https://lancedb.com/)에 의해 구동되는 내장된 개인 벡터 데이터베이스를 제공합니다. 기본 옵션을 사용할 때 벡터는 TeamplGPT를 떠나지 않습니다.

TeamplGPT는 여러 벡터 데이터베이스 제공자를 기본적으로 지원합니다.

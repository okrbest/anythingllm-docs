# 작업 공간 채팅 로그

**알고 계셨나요?** TeamplGPT는 채팅 로그를 올바른 형식으로 내보내 GPT-3.5 및 OpenAI의 다른 사용 가능한 모델을 세부 조정할 수 있도록 합니다.

TeamplGPT는 다음 형식으로 채팅을 내보내기를 지원합니다:

- **CSV**
- **JSON**
- **JSON (Alpaca)**
- **JSONL (OpenAI 세부 조정)**

채팅 로그가 최소 10개 이상 있으면 화면 상단의 내보내기 버튼을 클릭하세요! 올바른 계정 권한이 있으면 TeamplGPT 인스턴스의 작업 공간 및 사용자별로 채팅 로그를 볼 수 있습니다.

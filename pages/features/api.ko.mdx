# API 접근 및 키

사용 가능한 엔드포인트에 대한 API 문서는 인스턴스의 `/api/docs`에서 찾을 수 있습니다.

API 키는 올바른 접근 수준을 가진 계정에 의해 관리됩니다.

그러나 API 키를 가진 사람은 누구나 TeamplGPT API를 사용할 수 있으므로 이 키를 어디에도 공유하거나 게시하지 마세요.

TeamplGPT는 작업 공간을 관리, 업데이트, 임베드 및 채팅할 수 있는 전체 개발자 API를 지원합니다.

권한이 있는 경우 API 키를 즉시 생성하고 삭제할 수 있습니다.

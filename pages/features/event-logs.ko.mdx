# 이벤트 로그

TeamplGPT의 이벤트 로그 페이지는 사용자가 애플리케이션 내에서 발생하는 다양한 이벤트를 조회하고 모니터링할 수 있도록 합니다.

<Image
  src="/images/features/event-logs/event-logs.png"
  height={1080}
  width={1920}
  quality={100}
  alt="TeamplGPT Event Logs"
/>

이 기능은 사용자 활동 및 시스템 관련 이벤트에 대한 통찰력을 제공합니다.

## 이벤트 유형

이벤트 로그 페이지는 다음과 같은 다양한 이벤트를 캡처합니다:

- 사용자 로그인 시도 (성공 및 실패)
- 사용자가 보낸 메시지
- 애플리케이션 설정 변경
- 문서 업로드

## 이벤트 세부 정보

이벤트 로그 페이지의 각 이벤트에는 이벤트 유형, 관련 사용자(해당되는 경우), 타임스탬프 및 이벤트 유형에 특정한 추가 세부 정보와 같은 관련 정보가 포함됩니다.

TeamplGPT 인스턴스를 모니터링하는 데 유용합니다.

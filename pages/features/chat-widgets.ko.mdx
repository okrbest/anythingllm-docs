# 임베디드 채팅 위젯

TeamplGPT는 간단한 `<script>` 태그를 사용하여 어떤 웹사이트에도 쉽게 통합할 수 있는 임베디드 채팅 위젯을 생성할 수 있습니다. 이러한 임베디드 채팅 위젯은 사용자가 웹사이트에서 직접 챗봇과 상호 작용할 수 있는 편리한 방법을 제공합니다.

## 구성 옵션

임베디드 채팅 위젯을 생성할 때 동작과 외관을 사용자 정의할 수 있는 여러 구성 옵션이 있습니다.

### 작업 공간

작업 공간 설정은 채팅 창이 기반할 작업 공간을 결정합니다. 선택한 작업 공간의 모든 기본값이 특정 구성 옵션에 의해 재정의되지 않는 한 상속됩니다.

### 허용된 채팅 방법

허용된 채팅 방법을 사용하여 챗봇의 동작 방식을 설정할 수 있습니다. 두 가지 옵션이 있습니다:

- **채팅**: 챗봇은 문맥에 상관없이 모든 질문에 응답합니다.
- **쿼리**: 챗봇은 작업 공간의 문서와 관련된 채팅에만 응답합니다.

### 도메인으로부터의 요청 제한

이 필터를 사용하여 지정된 목록 이외의 도메인에서 오는 모든 요청을 차단할 수 있습니다. 이 필드를 비워 두면 누구나 임베디드 채팅 위젯을 모든 사이트에서 사용할 수 있습니다.

### 하루 최대 채팅 수

이 임베디드 채팅 위젯이 24시간 동안 처리할 수 있는 채팅 수를 제한할 수 있습니다. 이 값을 0으로 설정하면 하루에 무제한 채팅이 가능합니다.

### 세션당 최대 채팅 수

이 임베디드 채팅 위젯을 사용하여 세션 사용자에게 24시간 동안 보낼 수 있는 채팅 수를 제한할 수 있습니다. 이 값을 0으로 설정하면 세션당 무제한 채팅이 가능합니다.

### 동적 모델 사용 활성화

동적 모델 사용을 활성화하면 기본 작업 공간을 재정의하여 선호하는 LLM 모델 설정을 허용합니다.

### 동적 LLM 온도 활성화

동적 LLM 온도를 활성화하면 기본 작업 공간을 재정의하여 LLM 온도 설정을 허용합니다.

### 프롬프트 재정의 활성화

프롬프트 재정의를 활성화하면 기본 작업 공간을 재정의하여 시스템 프롬프트 설정을 허용합니다.

## 새로운 기능:

- [x] LiteLLM 임베더 지원
- [x] VoyageAI 임베더 지원
- [x] UI 메뉴 및 아이콘 버튼에 툴팁 추가
- [x] Github 도구의 PAT 요구 사항 알림 개선
- [x] Gemini 안전 임계값 설정
- [x] Gemini 1.5 Flash 지원
- [x] RAG 전면 개편
- [x] 채팅 내보내기 버튼 스타일 개선
- [x] Bing 및 Serply 검색 API를 `@agent`에 지원
- [x] LLM 실행을 위한 프롬프트 편집 기능
- [x] LLM 응답을 직접 편집할 수 있는 기능
- [x] 문서 선택기에서 문서를 검색할 수 있는 기능 추가 (유사 문자열 검색)
- [x] 스레드는 속도 향상을 위해 LLM 상호작용 없이 프롬프트 기반으로 자동 이름 변경

## 수정 사항:

- 슬래시 명령어를 삭제할 수 없는 문제 수정
- HTML이 포함된 코드 스니펫의 클립보드 복사 기능 문제 수정
- 스레드 삭제를 위한 `CMD/CTRL`이 _토글_이 아닌 _누르기_로 변경됨
- Chroma 컬렉션 이름 요구 사항 패치

## 다음 예정 사항:

- `@agent` 기능을 CrewAI, AutoGen 등으로 확장
- 에이전트 UI 전면 개편

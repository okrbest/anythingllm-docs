# AI 에이전트 설정

### 1) 에이전트 구성 메뉴로 이동

작업 공간 설정을 열고 에이전트 구성 메뉴로 이동합니다.

<Image
  src="/images/anythingllm-setup/agent-configuration/configuration-menu.png"
  height={1080}
  width={1920}
  quality={100}
  alt="TeamplGPT Agent Configuration Menu"
/>

### 2) 에이전트의 LLM 선택

작업 공간 설정에서 에이전트가 사용할 LLM 제공자와 모델을 선택합니다. "작업 공간 에이전트 업데이트" 버튼을 클릭하지 않으면 설정이 저장되지 않습니다.

<Image
  src="/images/anythingllm-setup/agent-configuration/llm.png"
  height={1080}
  width={1920}
  quality={100}
  alt="TeamplGPT Agent LLM"
/>

작업 공간 에이전트 설정을 업데이트한 후 "에이전트 스킬 구성" 버튼을 클릭합니다.

<Image
  src="/images/anythingllm-setup/agent-configuration/configure-agent-skills-button.png"
  height={1080}
  width={1920}
  quality={100}
  alt="TeamplGPT Configure Agent Skills"
/>

<Callout type="error" emoji="️⚠️">
  **주의**

    모든 LLM 모델이 에이전트로 잘 작동하는 것은 아닙니다. 더 나은 응답을 위해서는 높은 양자화 모델을 사용할 필요가 있습니다.
    예: `Llama 3 8B 8Bit Quantization`은 에이전트로서 더 나은 응답을 제공합니다.

</Callout>

<Callout type="info" emoji="️ℹ️">
  **참고**

    TeamplGPT는 `@agent`와 함께 사용할 수 있는 많은 LLM 제공자를 지원하며, 지원되는 모든 LLM은 [여기](https://docs.useanything.com/features/language-models#supported-language-model-providers)에서 확인할 수 있습니다.

</Callout>

### 3) 에이전트 스킬 선택

에이전트가 사용할 스킬 또는 도구를 선택합니다. "기본"으로 표시된 스킬은 비활성화할 수 없는 기본 스킬입니다.

<Image
  src="/images/anythingllm-setup/agent-configuration/tools.png"
  height={1080}
  width={1920}
  quality={100}
  alt="TeamplGPT Agent Skills"
/>

<Callout type="info" emoji="️ℹ️">
  **참고**

    `RAG`, `Summarize Documents` 및 `Scrape Websites`와 같은 일부 도구는 기본적으로 활성화되어 있으며 비활성화할 수 없습니다.

</Callout>

### 4) 검색 제공자 선택 (선택 사항)

에이전트가 사용할 수 있는 도구 중 하나는 인터넷을 검색할 수 있는 `Web-Browsing`입니다.

현재 다음 검색 제공자를 지원합니다:

- [Google](https://www.google.com/)
- [Serper](https://serper.dev/)
- [Bing Search](https://azure.microsoft.com/en-us/pricing/details/cognitive-services/bing-entity-search-api/)
- [Serply](https://serply.io/)

<Callout type="info" emoji="️ℹ️">
  **참고**

    `Web-Browsing` 도구를 사용할 계획이 없다면 이 단계를 무시해도 됩니다.

</Callout>

<Image
  src="/images/anythingllm-setup/agent-configuration/search.png"
  height={1080}
  width={1920}
  quality={100}
  alt="TeamplGPT Agent Web Browsing"
/>

Google 검색 엔진을 사용하면 매일 최대 100회 무료 검색을 수행할 수 있습니다.

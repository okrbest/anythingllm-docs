# AI 에이전트

에이전트는 기본적으로 몇 가지 간단한 도구에 접근할 수 있는 LLM입니다. 우리는 이 영역에 더 많은 사용자 정의 기능을 곧 추가할 예정입니다.

모든 에이전트는 작업 공간 전체에서 동일한 도구를 공유하지만, `@agent`를 통해 호출된 작업 공간 내에서 작동합니다.

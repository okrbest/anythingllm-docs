# 베타 미리보기란 무엇인가요?

TeamplGPT의 베타 미리보기는 다음과 같은 버전 _또는_ 기능입니다:

- 활발히 개발 중인 기능
- 불안정하거나 테스트되지 않은 기능
- 완전히 완료되지 않았지만 커뮤니티의 피드백을 필요로 하는 기능

# 베타 미리보기 유형

TeamplGPT의 베타 미리보기는 여러 가지 방법으로 제공될 수 있습니다:

- 완전히 분리된 Docker 이미지로
- [데스크탑 애플리케이션](/installation/desktop/macos)의 특별 출판물로
- TeamplGPT 애플리케이션에 이미 존재하는 숨겨진 기능으로

# 베타 미리보기의 영향 이해하기

우리는 기능을 신중하게 제작하지만, 개발 중이거나 피드백을 받고자 할 때 TeamplGPT 사용자에게 비표준 기능을 적극적으로 제공하고자 합니다.

가능한 경우, 특정 기능의 특정 위험에 대해 미리 경고해 드리겠습니다.

# 베타 기능 사용 시 발생할 수 있는 위험

- TeamplGPT의 로컬 데이터베이스, 소스 파일, 저장된 문서 또는 데이터 저장소의 부분적 또는 전체적 데이터 손실
- LLM, 임베더 또는 타사 제공자의 사용 증가
- LLM, 임베더 또는 벡터 데이터베이스 제공자로 타사 제공자를 사용할 경우 비용 증가
- 장치의 리소스 사용량 증가
- 로컬 DB 또는 벡터 데이터베이스의 손상
- TeamplGPT의 처리되지 않은 버그, 예외 및 충돌

# 기능 미리보기를 활성화하는 방법

<Callout>
  이 절차를 따라도 아무 일도 일어나지 않으면, 현재 사용 중인 TeamplGPT 버전에서는 활성화된 미리보기가 없다는 의미입니다.
</Callout>

Docker, 데스크탑, 호스팅 형태의 TeamplGPT에서 기능 미리보기를 활성화하려면 왼쪽 사이드바에서 "렌치" 아이콘을 클릭하여 설정 페이지를 엽니다.

다음으로, 키보드의 `Command` 또는 `Control` 키를 3초 동안 누르고 있어야 합니다. 실험적 기능이 활성화되었다는 경고가 표시됩니다.

이제 기능 관리 페이지에 접근할 수 있으며, 경고 모달을 이해하고 수락한 후 실험적 기능을 관리할 수 있습니다.

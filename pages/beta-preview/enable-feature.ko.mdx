# 베타 미리보기의 함축 의미 이해하기

우리는 기능을 신중하게 제작하지만, 개발 중이거나 피드백을 받고자 할 때 AnythingLLM 사용자에게 비표준 기능을 적극적으로 제공하고자 합니다.

가능할 경우 특정 기능의 위험성에 대해 사전에 알려드리겠습니다.

# 베타 기능 사용 시 발생할 수 있는 위험

- AnythingLLM의 로컬 데이터베이스, 소스 파일, 저장된 문서 또는 데이터 저장소의 부분적 또는 전체적 데이터 손실
- LLM, Embedder 또는 서드 파티 제공자 사용 증가
- LLM, Embedder 또는 VectorDatabase의 제공자로 사용될 경우 서드 파티 제공자 비용 증가
- 장치의 자원 사용 증가
- 로컬 DB 또는 벡터 데이터베이스 손상
- AnythingLLM의 처리되지 않은 버그, 예외 및 충돌

# 기능 미리보기를 활성화하는 방법

<Callout>
  이 절차를 따라도 아무 일도 일어나지 않으면 현재 사용 중인 AnythingLLM 버전에는 활성 미리보기가 없는 것입니다.
</Callout>

_모든_ 형태의 AnythingLLM (도커, 데스크탑, 호스팅)에서 기능 미리보기를 활성화하려면 왼쪽 사이드바에서 "렌치" 아이콘을 클릭하여 설정 페이지를 엽니다.

다음으로, 키보드에서 `Command` (Mac) 또는 `Control` (Windows/Linux) 키를 3초 동안 누르고 있습니다. 실험적 기능이 활성화되었다는 알림이 표시될 것입니다.

이제 기능 관리 페이지에 접근할 수 있으며 경고 모달을 이해하고 수락한 후 실험적 기능을 관리할 수 있습니다.

# 채팅 인터페이스 개요

TeamplGPT의 채팅 인터페이스는 TeamplGPT를 사용할 때 대부분의 시간을 보내게 될 곳입니다. 따라서 기본 사항에 익숙해져야 합니다. 이 페이지에는 위 이미지에 없는 추가 아이콘이 있을 수 있습니다. TeamplGPT는 항상 개선되고 있기 때문입니다.

위 이미지는 복잡해 보일 수 있지만, 다른 인터페이스와 유사하여 곧 익숙해질 것입니다.

## 사용자 메시지

사용자 메시지는 사용자가 보낸 메시지입니다. 이 텍스트는 유사한 문서를 찾는 데 사용되며 LLM(대형 언어 모델)에게 전송됩니다.

### 동작

- 복사: 이 텍스트 상자의 내용을 복사합니다.
- 편집: 메시지를 편집하면 해당 지점부터 대화를 수정하고 자동으로 LLM에 재전송할 수 있습니다. **주의**: 이 경우 편집된 내용 아래의 모든 메시지가 잘립니다.
- 말하기: 운영 체제의 기본 텍스트-음성 모듈, OpenAI Voice 또는 11Labs 음성을 사용하여 텍스트를 말합니다.

## LLM 메시지

LLM 메시지는 현재 채팅 세션에서 활성 상태인 LLM의 응답입니다. 이 텍스트는 유사한 문서를 찾는 데 사용되며 향후 대화에서도 사용됩니다. 컨텍스트 창이 초과되면 기록은 자동으로 관리됩니다.

### 동작

- 복사: 이 텍스트 상자의 내용을 복사합니다.
- 편집: 메시지를 편집하면 LLM 메시지의 출력을 수정할 수 있습니다. 이는 프롬프트를 재전송하지 않으며 단순히 기록을 업데이트합니다.
- 재생성: 동일한 프롬프트와 기록으로 다시 프롬프트를 LLM에 전송하여 새 답변을 받습니다.
- 피드백 (좋아요 & 싫어요): 사용자에게 LLM 응답에 대한 정성적 피드백을 남길 수 있게 합니다. 피드백을 남기는 것은 **메시지 기록이나 향후 응답에 영향을 미치지 않습니다**. 피드백 메트릭은 채팅 로그를 내보내어 좋은 응답을 찾아 TeamplGPT 외부에서 미세 조정할 수 있도록 정렬하는 데 가장 유용합니다.

## 프롬프트 입력 컨트롤

- 슬래시 명령: `슬래시 명령`은 프롬프트에 표준 텍스트를 삽입하는 방법입니다. 기본적으로 텍스트 스니펫의 단축키입니다. 여기서 슬래시 명령을 만들고 관리할 수 있습니다.
  - 기본 슬래시 명령: `/reset`과 같은 특수 기능을 가진 핵심 팀이 만든 특별한 명령입니다.
- `@agent` 호출: 사용 가능한 모든 `@agent`와 그들의 스킬 세트를 봅니다. 프롬프트 시작 부분에 `@agent`를 사용하면 에이전트 세션이 시작됩니다. [에이전트에 대해 더 알아보기.](/agent/overview)
- 글꼴 크기: TeamplGPT 프로필의 기본 글꼴 크기를 설정합니다.
- 마이크: LLM 프롬프트에 음성 인식을 활성화합니다. **이 기능은 데스크탑에서는 사용할 수 없습니다.**

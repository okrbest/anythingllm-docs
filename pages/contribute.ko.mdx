# 기여하기

TeamplGPT와 이 문서를 개선하기 위한 커뮤니티의 기여를 환영하고 감사드립니다.

## 기여 방법

### 이슈 생성

버그를 발견하거나 새로운 기능에 대한 제안이 있거나 개선 사항을 논의하고 싶다면, GitHub 저장소에 이슈를 생성해 주세요.

제안하는 문제나 개선 사항을 명확하게 설명해 주시면 팀이 신속하게 검토할 것입니다.

### 풀 리퀘스트 (PR)

풀 리퀘스트를 통한 기여를 적극 권장합니다.

버그 수정, 새로운 기능 구현, 문서 개선 등 어떤 기여도 소중히 여깁니다.

### 튜토리얼

TeamplGPT에 대한 지식과 전문 지식을 공유하여 튜토리얼을 작성해 보세요.

튜토리얼은 사용자가 프로젝트를 더 잘 이해하고 기능을 효과적으로 사용하는 데 도움이 됩니다.

### Discord에서 참여

아이디어를 논의하고, 도움을 구하고, 다른 기여자 및 사용자와 협력하기 위해 Discord 커뮤니티에 참여하세요.

Discord에서 참여하는 것은 프로젝트 개발 상황을 업데이트하고 커뮤니티와 연결하는 좋은 방법입니다.

## 시작하기

오픈 소스 프로젝트나 TeamplGPT에 기여하는 것이 처음이라면 다음 단계를 따라 시작할 수 있습니다:

### 익숙해지기:

프로젝트의 목표, 아키텍처 및 기존 코드베이스를 이해하는 데 시간을 할애하세요. GitHub 저장소와 문서를 탐색하여 익숙해질 수 있습니다.

### 이슈 선택:

GitHub 이슈를 살펴보고 해결할 수 있는 작업을 찾아보세요.

### 문의하기:

도움이 필요하거나 질문이 있는 경우, Discord 또는 작업 중인 GitHub 이슈의 댓글에서 도움을 요청하는 것을 주저하지 마세요.

기여 과정 전반에 걸쳐 지원해 드리겠습니다.

### 기여 제출:

변경 사항을 완료하면 GitHub에서 풀 리퀘스트를 제출하세요. 변경 사항과 관련된 세부 정보를 포함하여 명확한 설명을 작성해 주세요.

TeamplGPT에 기여해 주셔서 감사합니다. 여러분의 지원은 이 프로젝트를 모든 사람에게 더 나은 프로젝트로 만드는데 큰 도움이 됩니다!

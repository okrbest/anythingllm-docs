# TeamplGPT 기본 음성 텍스트 변환 모델

<Callout type="info" emoji="️💡">
  **참고:**

  제한된 RAM 또는 CPU를 가진 기기에서 로컬 Whisper 모델을 사용할 경우 미디어 파일을 처리할 때 TeamplGPT가 멈출 수 있습니다.
  최소한 2GB의 RAM을 권장하며, 10MB 이하의 파일을 업로드하세요.
</Callout>

TeamplGPT는 내장된 음성 텍스트 변환 모델 [Xenova Whisper](https://huggingface.co/Xenova/whisper-small)을 포함하고 있으며, 첫 사용 시 자동으로 다운로드됩니다.

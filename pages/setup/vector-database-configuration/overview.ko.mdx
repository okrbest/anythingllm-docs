# 벡터 데이터베이스

벡터 데이터베이스는 시스템 전체에서 설정되며, LLM처럼 작업 공간별로 원자적으로 구성할 수 없습니다.

<Callout type="error" emoji="️‼️">
  **주의!**

  벡터 데이터베이스 간 "이동"을 방지해야 합니다. TeamplGPT는 이미 내장된 정보를 자동으로 옮기지 않습니다.

  다른 벡터 데이터베이스로 마이그레이션하려면 각 작업 공간에서 모든 문서를 삭제하고 다시 임베딩해야 합니다.
</Callout>

TeamplGPT는 많은 벡터 데이터베이스 제공자를 기본적으로 지원합니다.

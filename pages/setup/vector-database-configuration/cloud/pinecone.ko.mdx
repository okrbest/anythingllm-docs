# Pinecone 벡터 데이터베이스

[Pinecone](https://www.pinecone.io/)는 모든 규모에서 빠르고 사용하기 쉬운, 개발자들이 선호하는 벡터 데이터베이스입니다.

Pinecone은 수십억 개의 벡터를 낮은 지연 시간으로 처리하여 신속하고 필터링된 쿼리 결과를 제공합니다.

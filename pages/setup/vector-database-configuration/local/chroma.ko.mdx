# Chroma 벡터 데이터베이스

[Chroma](https://trychroma.com)는 어디에서나 쉽게 실행하고 호스팅할 수 있는 [오픈 소스](https://github.com/chroma-core/chroma) AI 네이티브 벡터 데이터베이스입니다.

TeamplGPT는 로컬 또는 클라우드에서 호스팅된 Chroma 인스턴스에 연결하여 임베딩을 자동으로 저장하고 검색할 수 있습니다.

## Chroma 벡터 데이터베이스에 연결하기

<Callout type="warning" emoji="️⚠️">
  **개발자 알림**

  Chroma는 임베딩을 자동으로 임베딩하거나 인덱싱하려면 [서버가 실행 중이어야 합니다](https://docs.trychroma.com/usage-guide#running-chroma-in-clientserver-mode).

  TeamplGPT는 설정된 임베딩 모델을 사용하며, Chroma의 내장 임베더를 사용하지 않습니다.
</Callout>

Chroma는 **설정**에서 언제든지 구성할 수 있습니다.

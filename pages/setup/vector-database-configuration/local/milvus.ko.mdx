# Milvus 벡터 데이터베이스

[Milvus](https://github.com/milvus-io/milvus)는 임베딩 유사성 검색과 AI 애플리케이션을 지원하기 위해 구축된 오픈 소스 벡터 데이터베이스입니다.

Milvus는 비정형 데이터 검색을 더 쉽게 만들고, 배포 환경에 관계없이 일관된 사용자 경험을 제공합니다.

## Milvus 벡터 데이터베이스에 연결하기

Milvus는 **설정**에서 언제든지 구성할 수 있습니다.

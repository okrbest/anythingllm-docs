# TeamplGPT 기본 임베더

<Callout type="warning" emoji="️⚠️">
  **알림!**

    이 임베딩 모델은 첫 임베딩 시 (25MB) 다운로드되며 **CPU에서 실행**됩니다. 프로세스가 병목 현상을 일으키지 않도록 최소 2GB의 RAM이 필요합니다.

</Callout>

TeamplGPT는 CPU에서 실행되는 내장 임베더 모델을 포함하고 있습니다.

이 모델은 주로 영어 문서에 대해 학습된 인기 있는 [all-MiniLM-L6-v2](https://huggingface.co/sentence-transformers/all-MiniLM-L6-v2) 모델입니다.

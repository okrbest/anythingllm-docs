# OpenAI 임베더

OpenAI는 성능과 차원이 다른 3가지 임베딩 모델을 제공합니다. 최신 가격 정보는 OpenAI에 확인하십시오.

TeamplGPT에서 문서를 임베드하려고 할 때 비용 추정치를 제공합니다.

| 모델                   | 달러 당 ~ 페이지  | 최대 입력 |
| :--------------------- | :---------------: | :-------: |
| text-embedding-3-small |      62,500       |   8,191   |
| text-embedding-ada-002 |      12,500       |   8,191   |
| text-embedding-3-large |       9,615       |   8,191   |

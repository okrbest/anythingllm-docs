# 대형 언어 모델

대형 언어 모델은 방대한 양의 텍스트 데이터를 처리하여 인간 언어를 이해하고 생성할 수 있는 AI 시스템입니다.

## TeamplGPT의 LLM 유형

TeamplGPT는 LLM을 구체적으로 또는 일반적으로 선택하여 설정할 수 있으며, 동일한 애플리케이션 내에서 여러 LLM을 동시에 구성할 수도 있습니다.

### 시스템 LLM

시스템 LLM은 TeamplGPT가 기본적으로 사용하는 LLM입니다. 워크스페이스나 에이전트에 특정 LLM이 정의되지 않은 경우, 이 시스템 LLM이 사용됩니다.

### 워크스페이스 LLM

TeamplGPT는 워크스페이스별로 특정 LLM을 설정할 수 있습니다. 이렇게 설정된 워크스페이스 LLM은 시스템 LLM을 재정의하지만, 이는 해당 워크스페이스와 채팅할 때만 적용됩니다. 이를 통해 각 워크스페이스마다 고유한 제공자, 모델 또는 둘 다를 사용할 수 있습니다.

### 에이전트 LLM

TeamplGPT는 AI 에이전트를 지원합니다. 에이전트의 경우, 모든 LLM이 동일하게 만들어진 것은 아닙니다. 일부 LLM은 더 나은 AI 에이전트 기능을 위해 도구 호출을 직접 지원합니다. 이 모델은 에이전트와 함께 사용하기 위해 명시적으로 사용되는 모델입니다.

## 지원되는 LLM 제공자

TeamplGPT는 최소한의 설정으로 여러 LLM을 기본적으로 지원합니다.

LLM은 워크스페이스나 에이전트가 질문과 프롬프트에 어떻게 응답하는지를 결정하는 기본 통합입니다.

TeamplGPT 내에서 언제든지 LLM 제공자, 모델 또는 기타 세부 사항을 수정할 수 있습니다.

로컬 및 클라우드 기반 LLM 모두에 연결할 수 있습니다 - 심지어 동시에!

# TeamplGPT 기본 LLM

<Callout type="info" emoji="️💡">
  **데스크탑 전용!**

    이 기본 LLM 제공자 기능은 TeamplGPT의 데스크탑 버전에서만 제공됩니다.

</Callout>

TeamplGPT는 LLama-3, Phi-3 등 인기 있고 높은 평가를 받은 LLM을 로컬 CPU 및 GPU에서 실행할 수 있도록 내장된 LLM 엔진과 제공자를 포함합니다.

TeamplGPT 데스크탑을 부팅하면 다운로드하려는 모델을 선택할 수 있습니다. 진행 상황은 응용 프로그램 창의 오른쪽 상단에 표시됩니다.

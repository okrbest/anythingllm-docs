# Mistral AI LLM

[Mistral AI](https://mistral.ai/)는 인기 있는 검열되지 않은 오픈 소스 **Mistral-7B** 모델의 제작자입니다.

이들은 채팅에 사용할 수 있는 간단한 인터페이스의 API를 제공합니다.

## Mistral AI 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키 필요!**

    이 통합을 사용하려면 [mistral.ai](https://mistral.ai/)에서 유효한 API 키를 얻어야 합니다.

</Callout>

<Callout type="info" emoji="️💡">
  **알림!**

    API 기반 모델은 민감한 주제에 대해 검열될 수 있으며, 오픈 소스 모델은 검열되지 않습니다.

    전체 모델을 사용하려면 [LocalAI](/llms/localai), [LMStudio](/llms/lmstudio), 또는 [Ollama](/llms/ollama)와 같은 로컬 LLM 제공자를 사용하십시오.

</Callout>

다른 LLM 제공자와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

모든 Mistral 모델은 현재 TeamplGPT에서 사용 가능합니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

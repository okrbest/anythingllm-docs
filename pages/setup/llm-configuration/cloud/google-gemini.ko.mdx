# Google Gemini LLM

[Google Gemini Pro](https://ai.google.dev/)는 GPT와 동등한 응답을 제공하는 모델로 현재 무료로 사용할 수 있습니다. API 키에 등록하기만 하면 됩니다.

## Google Gemini 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키 필요!**

    이 통합을 사용하려면 [ai.google.dev](https://ai.google.dev/)에서 유효한 API 키를 얻어야 합니다.

</Callout>

다른 LLM 제공자와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

현재 모든 Google Gemini 모델을 TeamplGPT에서 사용할 수 있습니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

# Cohere LLM

[Cohere](https://cohere.com/)는 실질적인 문제를 해결하는 기업용 사례의 요구를 충족시키기 위해 업계 선도적인 대형 언어 모델(LLM)과 RAG 기능을 제공합니다.

## Cohere 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키 필요!**

    이 통합을 사용하려면 [Cohere.com](https://cohere.com/)에서 유효한 API 키를 얻어야 합니다.

</Callout>

다른 LLM 제공자와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

현재 모든 Cohere 모델을 TeamplGPT에서 사용할 수 있습니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

# Anthropic LLM

[Anthropic](https://Anthropic.com)은 훨씬 더 큰 컨텍스트 윈도우와 고성능을 자랑하는 Claude-3와 같은 모델을 호스팅하는 것으로 유명한 모델 제공업체입니다.

## Anthropic 연결

<Callout type="info" emoji="️💡">
  **유효한 API 키 필요!**

    이 통합이 작동하려면 [console.anthropic.com](https://console.anthropic.com/)에서 유효한 API 키를 얻어야 합니다.

</Callout>

다른 LLM 제공업체와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

현재 모든 Anthropic 모델을 TeamplGPT와 함께 사용할 수 있습니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

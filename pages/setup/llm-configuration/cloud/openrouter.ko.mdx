# OpenRouter LLM

[OpenRouter](https://openrouter.ai)는 모든 오픈 소스 LLM 모델을 간편하게 호스팅하는 인기 있는 모델 제공자입니다.

새로운 모델이 출시되면 가장 먼저 OpenRouter에 나타납니다.

## OpenRouter에 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키가 필요합니다!**

  이 통합 기능을 사용하려면 [openrouter.ai](https://openrouter.ai/)에서 유효한 API 키를 받아야 합니다.
</Callout>

다른 LLM 제공자와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

모든 OpenRouter 모델은 현재 TeamplGPT에서 사용할 수 있습니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

# Groq LLM

[Groq](https://groq.com)은 오픈 소스 모델을 가장 빠르게 실행하는 방법을 선도하는 모델 제공자로 유명합니다.

이 제공자는 LLM으로부터 거의 즉각적인 응답을 받을 수 있게 해줍니다.

속도가 가장 중요한 경우, Groq는 비교할 수 없는 선택입니다.

## Groq 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키 필요!**

    이 통합을 사용하려면 [Groq AI](https://wow.groq.com/)에서 유효한 API 키를 얻어야 합니다.

</Callout>

다른 LLM 제공자와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

현재 모든 Groq 모델을 TeamplGPT에서 사용할 수 있습니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

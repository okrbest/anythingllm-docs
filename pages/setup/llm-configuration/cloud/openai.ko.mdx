# OpenAI LLM

[OpenAI](https://OpenAI.com)은 많은 TeamplGPT 사용자들에게 가장 인기 있는 비공개 소스 옵션입니다.

시스템, 워크스페이스, 에이전트 실행을 위한 현재의 모든 채팅 모델을 지원합니다.

## OpenAI 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키 필요!**

    이 통합을 사용하려면 [platform.openai.com](https://platform.openai.com)에서 유효한 API 키를 얻어야 합니다.

    또한 결제 계정을 연결해야 이 제공자를 사용할 수 있습니다.

</Callout>

모든 OpenAI 모델은 현재 TeamplGPT에서 사용 가능합니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

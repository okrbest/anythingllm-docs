# Together AI LLM

[Together AI](https://www.together.ai/)는 별도의 인프라를 구축하지 않고도 수백 가지의 다양한 오픈 소스 모델에 대한 API 액세스를 제공하는 온라인 서비스입니다.

## Together AI에 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키가 필요합니다!**

  이 통합 기능을 사용하려면 [Together.ai](https://www.Together.ai/)에서 유효한 API 키를 받아야 합니다.
</Callout>

다른 LLM 제공자와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

모든 Together AI 모델은 현재 TeamplGPT에서 사용할 수 있습니다. [지원되는 전체 모델 목록 보기.](https://docs.together.ai/docs/inference-models#chat-models)

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

# Perplexity AI LLM

[Perplexity AI](https://www.perplexity.ai/)는 항상 최신의 응답을 제공하는 '인터넷 지원' 모델로 유명한 모델 제공자입니다. 최신 공공 정보를 위해 RAG가 필요하지 않습니다.

## Perplexity AI에 연결하기

<Callout type="info" emoji="️💡">
  **유효한 API 키가 필요합니다!**

  이 통합 기능을 사용하려면 [perplexity.ai](https://www.perplexity.ai/)에서 유효한 API 키를 받아야 합니다.
</Callout>

다른 LLM 제공자와 마찬가지로, API 키를 입력하면 채팅 모델 선택 드롭다운이 자동으로 채워집니다.

모든 Perplexity AI 모델은 현재 TeamplGPT에서 사용할 수 있습니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

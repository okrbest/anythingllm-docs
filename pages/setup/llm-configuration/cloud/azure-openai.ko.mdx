# Azure OpenAI LLM

Microsoft Azure OpenAI는 기본 [OpenAI 제공자](openai)와 동일한 LLM 모델을 제공하지만, 모든 개인정보 보호 및 해당 구독과 관련된 계약을 포함하여 Azure 계정에서 실행됩니다.

## Azure OpenAI 연결

<Callout type="info" emoji="️💡">
  **유효한 계정 설정 필요!**

    이 통합을 사용하려면 유효한 [Azure OpenAI](https://azure.microsoft.com/en-us/products/ai-services/openai-service) 구독이 설정되어 있어야 합니다.

</Callout>

Microsoft Azure를 사용하여 LLM 채팅 모델을 사용할 수 있습니다.

이를 통해 Microsoft에서 관리하는 프라이빗 및 엔터프라이즈 환경에서 GPT 모델을 사용할 수 있습니다.

**설정**에서 언제든지 다른 모델로 업데이트할 수 있습니다.

# OpenAI (Generic) LLM

<Callout type="error" emoji="️‼️">
  **주의!**

    이 제공자는 개발자를 위한 것이며, 무엇을 하고 있는지 잘 모르면 사용하지 않는 것이 좋습니다.

</Callout>

`Generic` OpenAI 래퍼는 우리가 명시적으로 통합하지 않은 모든 LLM 제공자와 쉽게 상호 작용할 수 있는 방법으로, API 기능과 추론 응답 모두에서 `OpenAI 호환`을 제공합니다.

이 제공자를 사용하려면 상호 작용하려는 LLM 제공자가 OpenAI와 호환되는지 확인하고 각 입력의 용도를 이해해야 합니다.

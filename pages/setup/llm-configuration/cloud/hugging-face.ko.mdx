# HuggingFace LLM

[HuggingFace](https://huggingface.co)는 전 세계의 오픈 소스 LLM 및 기타 AI 모델을 온라인에 올리는 곳입니다.

100,000개 이상의 LLM의 모든 원시 모델 파일을 여기에서 찾을 수 있으며, TeamplGPT에 연결하여 실행할 수 있습니다.

## Hugging Face 연결하기

<Callout type="info" emoji="️💡">
  **유효한 구성 필요!**

    이 통합은 HuggingFace가 운영하는 서버리스 추론 서비스에 특정됩니다.

</Callout>

  **팁:**

    이 통합은 모델의 채팅 템플릿이 정의된 경우에 가장 잘 작동합니다. 그렇지 않으면 예상치 못한 결과가 나올 수 있습니다.

</Callout>

## 설치 개요

TeamplGPT는 Mac, Windows 또는 Linux 운영 체제에서 설치할 수 있는 "**싱글 플레이어**" 애플리케이션으로, 최소한의 설정으로 로컬 LLM, RAG 및 에이전트를 사용할 수 있으며 완전한 개인정보 보호를 제공합니다.

TeamplGPT는 데스크탑 애플리케이션으로 설치하거나, Docker를 사용하여 로컬에서 자체 호스팅하거나, 클라우드(AWS, 구글 클라우드, Railway 등)를 사용하여 호스팅할 수 있습니다.

## Docker vs 데스크탑 버전

TeamplGPT는 두 가지 주요 사용 방법을 제공합니다. 각 방법의 기능에는 몇 가지 뚜렷한 차이점이 있습니다. 두 방법 모두 오픈 소스입니다.

### TeamplGPT 데스크탑이 필요한 경우...

- 로컬에서 로컬 LLM, RAG 및 에이전트를 사용할 수 있는 원클릭 설치 앱이 필요할 때
- 다중 사용자 지원이 필요하지 않을 때
- 모든 것이 장치에만 있어야 할 때
- 공개 인터넷에 아무 것도 "게시"할 필요가 없을 때 (예: 웹사이트용 채팅 위젯)

### TeamplGPT Docker가 필요한 경우...

- 로컬 LLM, RAG 및 에이전트를 사용할 수 있는 서버 기반 서비스를 원할 때
- 여러 사람이 동시에 사용할 수 있는 TeamplGPT 인스턴스를 실행하려 할 때
- 초대된 사용자와 인스턴스에서 정보를 공유하려 할 때
- 작업 공간 및 문서에 대한 관리자 및 규칙 기반 접근이 필요할 때
- 공개 인터넷에 채팅 위젯을 게시하려 할 때
- 브라우저에서 TeamplGPT에 접근하려 할 때

아래 표는 플랫폼 간 지원되는 기능의 비포괄적인 목록입니다.

| 기능                            | 데스크탑에서 사용 가능 | Docker에서 사용 가능 |
| :------------------------------ | :-------------------: | :-----------------: |
| 다중 사용자 지원                |          ❌           |         ✅          |
| 임베디드 채팅 위젯              |          ❌           |         ✅          |
| 원클릭 설치                     |          ✅           |         ❌          |
| 개인 문서                       |          ✅           |         ✅          |
| 모든 벡터 데이터베이스에 연결   |          ✅           |         ✅          |
| 모든 LLM 사용                   |          ✅           |         ✅          |
| 내장 임베딩 제공자              |          ✅           |         ✅          |
| 내장 LLM 제공자                 |          ✅           |         ❌          |
| 화이트 라벨링                   |          ❌           |         ✅          |
| 채팅 로그                       |          ✅           |         ✅          |
| 에이전트 지원                   |          ✅           |         ✅          |
| 에이전트 기술                   |          ✅           |         ✅          |
| 타사 데이터 커넥터              |          ✅           |         ✅          |
| 비밀번호 보호                   |          ❌           |         ✅          |
| 인스턴스에 새로운 사용자 초대   |          ❌           |         ✅          |
| 텍스트 분할 구성                |          ✅           |         ✅          |
| Whisper 모델 지원               |          ✅           |         ✅          |
| 전체 개발자 API                 |          ✅           |         ✅          |
| 사용자 관리                     |          ❌           |         ✅          |
| 작업 공간 접근 관리             |          ❌           |         ✅          |
| 웹사이트 스크래핑               |          ✅           |         ✅          |

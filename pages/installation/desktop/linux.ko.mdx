# Linux 설치

Linux에 TeamplGPT를 설치하는 방법은 **두 가지**가 있습니다.

**[1. 설치 `.AppImage` 파일을 사용하여 설치](#install-using-the-installation-file)**

**[2. Docker를 사용하여 설치](#install-using-docker)**

<Callout type="warning" emoji="⚠️">
  **운영 체제 공지**

➤ TeamplGPT는 Ubuntu 시스템에서 빌드되었으며 다른 배포판에서는 잘 작동하지 않을 수 있습니다.

➤ 설치 또는 부팅 문제 발생 시 [Github Issue](https://github.com/Mintplex-Labs/anything-llm/issues)를 열어주세요.

➤ 모든 Linux OS에서 TeamplGPT와 동일한 경험을 제공하고자 합니다.

</Callout>

## 애플리케이션 설치

Linux용 TeamplGPT `.AppImage`를 다운로드한 후 파일에 실행 권한을 부여해야 실행할 수 있습니다.

```shell copy "
 chmod a+x /Downloads/AnythingLLMDesktop.AppImage
```

그 후, 다른 앱 이미지처럼 AppImage를 부팅할 수 있습니다. 더블 클릭하거나 터미널을 통해 실행하면 전체 애플리케이션이 부팅됩니다.

```shell copy showLineNumbers"
# Assuming we are running the AppImage from the /Downloads folder.
./AnythingLLMDesktop.AppImage
# This will run the application in this terminal pane. It will show all logs.
```

## Docker를 사용하여 설치

[Docker](https://www.docker.com/)가 장치에 설치되어 있는지 확인하세요. Docker가 설치되어 있지 않다면 이 [가이드](https://docs.docker.com/desktop/install/linux-install/)를 따라 설치할 수 있습니다.

터미널에서 다음 명령어를 실행하세요.

```shell copy
 docker pull mintplexlabs/anythingllm
```

이 명령어는 Docker에서 최신 TeamplGPT 이미지를 가져옵니다.

최신 이미지를 가져온 후, 다음 명령어를 실행하세요.

```shell copy showLineNumbers
 export STORAGE_LOCATION=$HOME/anythingllm && \
 mkdir -p $STORAGE_LOCATION && \
 touch "$STORAGE_LOCATION/.env" && \
 docker run -d -p 3001:3001 \
 --cap-add SYS_ADMIN \
 -v ${STORAGE_LOCATION}:/app/server/storage \
 -v ${STORAGE_LOCATION}/.env:/app/server/.env \
 -e STORAGE_DIR="/app/server/storage" \
 mintplexlabs/anythingllm
```

위 명령어는 로컬 저장소를 마운트하고 Docker에서 TeamplGPT를 실행합니다.

이제 브라우저에서 `http://localhost:3001`로 이동하면 TeamplGPT를 사용할 수 있습니다!

모든 데이터와 진행 상황은 컨테이너 재빌드 또는 Docker Hub에서 가져오기 간에 지속됩니다.
